ZX-Poly streaming

The page is hosted by ZX-Poly emulator and contains link to its active video stream.

ZX-Poly emulator is the official OSS emulator of ZX-Poly platform (a multi-CPU ZX-Spectrum 128 clone). The project page hosted by GitHub.

Video-stream URL: ${video.link}
Mime-type: ${video.mime}

Tips for some media players

FFplay

Project page:https://ffmpeg.org/

ffplay -fflags nobuffer -flags low_delay ${video.link}

XBMC/KODI

Project page:https://kodi.tv/

curl -X POST -H "Content-Type: application/json" -d '{"jsonrpc":"2.0","method":"Player.Open","params":{"item":{"file":"${video.link}"}},"id":1}' http://KODI_HTTPCONTROL_ADDRESS_PORT/jsonrpc

MPV

Project page:https://mpv.io/

mpv --no-cache --untimed --no-demuxer-thread --video-sync=audio --vd-lavc-threads=1 ${video.link}

VLC

Project page:https://www.videolan.org/

vlc --network-caching=0 --tcp-caching=0 ${video.link}
